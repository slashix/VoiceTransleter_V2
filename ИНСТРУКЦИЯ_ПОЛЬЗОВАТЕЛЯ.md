# Инструкция пользователя VoiceTransleter V2

## О программе

**VoiceTransleter V2** — десктопное приложение для автоматического дубляжа видео. Программа распознаёт речь на любом из 27 языков, переводит её на русский и синтезирует голос с клонированием по аудиообразцу, сохраняя оригинальные тайминги видео.

**Версия:** 2.1.7  
**Разработчик:** Андраник Алавердян (AndranikFutureLabs)  
**Поддержка:** [@AndranikFutureLabs](https://t.me/AndranikFutureLabs)

**GitHub:** https://github.com/AndranikFutureLabs/VoiceTransleter_V2  
**Релизы:** https://github.com/AndranikFutureLabs/VoiceTransleter_V2/releases

## 1. Установка

### 1.1. Windows

#### Вариант 1: Установщик (рекомендуется)

1. Скачайте `VoiceTransleter.V2.Setup.2.1.7.exe` со страницы [релизов](https://github.com/AndranikFutureLabs/VoiceTransleter_V2/releases/tag/v2.1.7)
2. Запустите установщик
3. Если Windows показывает предупреждение «Система Windows защитила ваш компьютер» (Синий экран SmartScreen):
   - Нажмите **«Подробнее»**
   - Нажмите **«Выполнить в любом случае»**
   - Это нормально — приложение не имеет цифровой подписи (не куплен сертификат)
4. Выберите папку установки (или оставьте по умолчанию)
5. После установки запустите программу через ярлык на рабочем столе или меню Пуск

#### Вариант 2: Сборка из исходников

```bash
git clone https://github.com/AndranikFutureLabs/VoiceTransleter_V2.git
cd VoiceTransleter_V2
npm install
pip install faster-whisper TTS
npm run dev        # запуск в режиме разработки
npm run dist       # сборка установщика
```

### 1.2. macOS

**Intel:** скачайте `VoiceTransleter.V2-2.1.7.dmg`  
**Apple Silicon (M1/M2/M3):** скачайте `VoiceTransleter.V2-2.1.7-arm64.dmg`

1. Откройте `.dmg` файл
2. Перетащите VoiceTransleter V2 в папку Applications
3. При первом запуске: ПКМ по приложению → «Открыть» (для обхода Gatekeeper)

### 1.3. Linux

**AppImage (все дистрибутивы):**

```bash
chmod +x VoiceTransleter.V2-2.1.7.AppImage
./VoiceTransleter.V2-2.1.7.AppImage
```

> Для AppImage может потребоваться `sudo apt install libfuse2` (Debian/Ubuntu)

**DEB (Debian/Ubuntu):**

```bash
sudo dpkg -i voice-transleter_2.1.7_amd64.deb
sudo apt install -f   # если не хватает зависимостей
```

> **Важно:** для работы на macOS и Linux требуется установленный **FFmpeg**:
> - macOS: `brew install ffmpeg`
> - Linux: `sudo apt install ffmpeg`

## 3. Первый запуск

### 3.1. Загрузка моделей

При первом запуске приложение проверяет готовность компонентов. В строке статуса вы увидите три индикатора:

| Индикатор | Что означает |
|---|---|
| **FFmpeg** | Найден ли FFmpeg в системе или в папке приложения |
| **Whisper** | Загружена ли модель распознавания речи |
| **TTS** | Загружена ли модель синтеза речи |

Если модели не загружены, нажмите кнопку **«Загрузить модели»** (оранжевая). Будут скачаны:

- **faster-whisper medium** (~1.5 ГБ) — распознавание речи
- **XTTS v2** (~2.5 ГБ) — синтез речи с клонированием голоса

Загрузка происходит один раз. Модели кэшируются и при следующих запусках загружаются из кэша за несколько секунд.

### 3.2. Установка FFmpeg (если не найден)

**Windows:** FFmpeg уже встроен в установщик (файлы в папке `resources/`). Если индикатор красный — скачайте [FFmpeg](https://ffmpeg.org/download.html) и поместите `ffmpeg.exe` и `ffprobe.exe` в папку установки приложения.

**macOS:** `brew install ffmpeg`  
**Linux:** `sudo apt install ffmpeg`

## 4. Использование

### 4.1. Выбор видео

1. Нажмите на центральную область «Нажмите для выбора видеофайла» или перетащите файл мышью
2. Поддерживаемые форматы: **MP4, AVI, MKV, MOV, WebM**
3. После выбора область станет зелёной

### 4.2. Выбор языка оригинала

В выпадающем списке «Язык оригинала» выберите язык речи в видео или оставьте **«Автоопределение»**.

Поддерживаемые языки: английский, испанский, французский, немецкий, итальянский, португальский, нидерландский, польский, русский, китайский, японский, корейский, арабский, турецкий, хинди, вьетнамский, тайский, украинский, шведский, датский, финский, чешский, румынский, венгерский, греческий, иврит, индонезийский.

### 4.3. Выбор голоса

В выпадающем списке «Голос дубляжа» выберите один из доступных голосов.

### 4.4. Добавление своего голоса

1. Нажмите кнопку **«+»** рядом со списком голосов
2. Выберите WAV-файл с образцом голоса (10–30 секунд чистой речи без шумов)
3. Введите название для нового голосового профиля
4. Голос появится в списке и будет сохранён для будущих сессий

### 4.5. Запуск дубляжа

1. Убедитесь, что видео выбрано, а все три индикатора статуса зелёные
2. Нажмите **«Начать дубляж»**
3. Процесс отображается в логе и прогресс-баре в реальном времени:

| Шаг | Описание |
|---|---|
| **1/5** | Извлечение аудио из видео (FFmpeg) |
| **2/5** | Распознавание речи (faster-whisper, модель medium) |
| **3/5** | Перевод на русский (Google Translate) |
| **4/5** | Синтез речи (XTTS v2 — самый долгий шаг) |
| **5/5** | Сборка финального видео (FFmpeg) |

4. После завершения появится кнопка **«Открыть папку с результатами»**

## 5. Результаты

После завершения дубляжа в папке результатов создаются следующие файлы:

| Файл | Описание |
|---|---|
| `*_dubbed.mp4` | Готовое видео с русской озвучкой |
| `*_source.txt` | Исходный текст (формат SRT с таймингами) |
| `*_source_plain.txt` | Исходный текст без таймингов |
| `*_translation.txt` | Перевод на русский (SRT) |
| `*_translation_plain.txt` | Перевод без таймингов |
| `*_source_translit.txt` | Транслитерация оригинала (SRT) |
| `*_source_translit_plain.txt` | Транслитерация оригинала без таймингов |
| `*_translation_translit.txt` | Транслитерация перевода (SRT) |
| `*_translation_translit_plain.txt` | Транслитерация перевода без таймингов |

### Где хранятся результаты?

| ОС | Путь |
|---|---|
| **Windows** | `%APPDATA%\voice-transleter\output\` |
| **macOS** | `~/Library/Application Support/voice-transleter/output/` |
| **Linux** | `~/.config/voice-transleter/output/` |

## 8. Устранение неполадок

**В: Программа не запускается, ошибка «FFmpeg не найден»**  
О: На Windows FFmpeg встроен в установщик. Если ошибка persistирует — скачайте FFmpeg и поместите `ffmpeg.exe` и `ffprobe.exe` в папку установки. На macOS/Linux установите через `brew install ffmpeg` или `sudo apt install ffmpeg`.

**В: Модели не загружаются**  
О: Проверьте интернет-соединение. Модели весят ~4 ГБ. При медленной загрузке можно использовать зеркало Hugging Face:
```bash
# Windows (PowerShell)
$env:HF_ENDPOINT="https://hf-mirror.com"

# macOS / Linux
export HF_ENDPOINT=https://hf-mirror.com
```

**В: Долго обрабатывается видео**  
О: Самый долгий этап — синтез речи (XTTS v2 на CPU). RTF ~3.0 (1 секунда речи ≈ 3 секунды синтеза). Для ускорения используйте более короткие видео (до 5 минут) или запускайте на машине с большим количеством ядер.

**В: Качество распознавания низкое**  
О: Используйте видео с чистым звуком. Укажите язык оригинала вручную вместо «Автоопределение», если язык известен.

**В: Голос звучит неестественно**  
О: Используйте WAV-образец длительностью 10–30 секунд с чистой речью без музыки и шумов. Формат WAV даёт лучший результат, чем MP3.

**В: Не сохраняются текстовые файлы**  
О: Текстовые файлы сохраняются в папке результатов рядом с готовым видео. Нажмите «Открыть папку с результатами» после завершения.

**В: На macOS пишет «приложение повреждено»**  
О: ПКМ по приложению → «Открыть» → подтвердите. Это связано с отсутствием подписи разработчика (приложение собрано без кодовой подписи).

**В: На Linux AppImage не запускается**  
О: Убедитесь, что файл исполняемый (`chmod +x`) и установлен FUSE: `sudo apt install libfuse2`.
